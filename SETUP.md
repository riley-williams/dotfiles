# Setup

requires nvim nightly (0.9+)

## Required dependencies

swift, rust
macOS:
- `brew install ripgrep`
- `brew install nvm`

Linux:
- `sudo apt install nvm`
- swift, rust...

# Initial install

`nvim` - may have errors, that's ok
`:PlugInstall`
exit nvim and re-open, should see lsp support downloaded
